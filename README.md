# QuickTap Connect
### Dell InnovateDash 2026 – Republic Polytechnic

QuickTap Connect is a mobile and cloud solution that helps Dell booth representatives
capture, enrich, and route event leads using NFC technology.

## 📱 Features
- Tap-to-capture lead info using NFC badges
- Offline data storage with auto-sync to cloud
- Automatic routing to correct Dell team based on interest area
- Secure and scalable backend using cloud-native best practices

## 🧠 Tech Stack
- **Frontend:** Flutter (Mobile)
- **Backend:** Node.js (Express) + Firebase / MongoDB
- **Cloud:** AWS / Azure
- **Version Control:** GitHub
